一、第一次在github上新建仓库时采用的：

1.在本地某个文件夹下新建某个空库       git init

2.增加当前文件夹下的某个文件到上传列表  git add 程序

3.git commit -m "文件说明"

4.git remote add origin(随便写) https://github.com/14030140108/LeetCode(仓库地址)

5.git push -u origin master  //上传


二、更新代码时采用的：

1.git add 某个文件

2.git commit -m "说明"

3.git push -u origin master   //上传，这里的origin就是第一次使用的时候的别名
